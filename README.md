Wrappers
========

slcurl++
--------

A c++ wrapper for [curl](https://github.com/curl/curl).

slgit2++
--------

A c++ wrapper for [libgit2](https://github.com/libgit2/libgit2).

slpcre2++
---------

A c++ wrapper for [pcre2](https://github.com/PCRE2Project/pcre2).

slsqlite++
----------

A c++ wrapper for [sqlite3](https://github.com/sqlite/sqlite).

Helpers
=======

slcves++
--------

A c++ library to deal with CVEs.

slhelpers++
-----------

A c++ library to support colors, directories, environment variables, processes,
SSH keys, strings and much more.

slkerncvs++
-----------

A c++ library to deal with SUSE's kerncvs.
